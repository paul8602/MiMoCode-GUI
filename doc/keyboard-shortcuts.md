# Keyboard Shortcuts

## Global (System-wide)

| Shortcut | Action |
|----------|--------|
| `Cmd+Shift+M` | Show/Hide window |
| `Cmd+Shift+N` | New session |

## Navigation

| Shortcut | Action |
|----------|--------|
| `Cmd+B` | Toggle sidebar |
| `Cmd+Shift+B` | Toggle summary panel |
| `Cmd+O` | Open project |
| `Cmd+Alt+↑` | Previous project |
| `Cmd+Alt+↓` | Next project |
| `Alt+↑` | Previous session |
| `Alt+↓` | Next session |
| `Shift+Alt+↑` | Previous unseen session |
| `Shift+Alt+↓` | Next unseen session |

## Session

| Shortcut | Action |
|----------|--------|
| `Cmd+N` | New session |
| `Cmd+Shift+Backspace` | Archive session |
| `Cmd+.` | Cycle agent forward |
| `Cmd+Shift+.` | Cycle agent backward |
| `Enter` | Send message |
| `Shift+Enter` | New line in composer |
| `Escape` | Abort current operation |

## View

| Shortcut | Action |
|----------|--------|
| `Cmd+Shift+T` | Cycle theme |
| `Cmd+Shift+S` | Cycle color scheme |
| `Cmd+,` | Open settings |
| `Cmd+Shift+P` | Command palette |
| `Ctrl+`` | Toggle terminal |

## Review

| Shortcut | Action |
|----------|--------|
| `Cmd+Shift+B` | Toggle review panel |

## File

| Shortcut | Action |
|----------|--------|
| `Cmd+K` | Quick search |
